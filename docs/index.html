MRF-Based Multi-Omics Integration and Clustering

Tools for model-based integration of multi-omics datasets using random forests and Markov random field inspired weighting. The package provides workflows for model fitting, variable importance aggregation, clustering, and visualization.

1. Installation

Install the development version from GitHub:

> remotes::install_github("noblegasss/multiRF")

The package links to C++ via Rcpp for fast t-SNE computation and requires compilation tools (Rtools on Windows, Xcode CLI on macOS).

2. Example Data

The package ships a TCGA BRCA dataset with three matched omics blocks (gene expression, methylation, miRNA) and clinical annotations.

library(multiRF)
data("tcga_brca_data")  # loads tcga_brca and tcga_brca_clinical

names(tcga_brca)
dim(tcga_brca$gene)

#> [1] 674 200

Note. Samples must be aligned across all omics blocks. Each block is a matrix-like object with samples in rows and features in columns.

4. Clustering Workflows

The package provides two main entry points. mrf3() is the compact interface for standard workflows; mrf3_fit() exposes the full staged parameter surface.

4.1. Basic workflow with mrf3()

fit <- mrf3(
  tcga_brca,
  k = 4,
  ntree = 100,
  filter_mode = "none",
  run_imd = TRUE,
  seed = 529
)

summary(fit)
table(get_clusters(fit))
names(get_weights(fit))
get_top_vars(fit, n = 10)

4.2. Clustering backends

The global clustering backend is controlled by main_clustering. All three options are compared against PAM50 subtypes in Section 8.

Similarity (default) 
fit_similarity <- mrf3(
  tcga_brca, k = 4,
  main_clustering = "similarity",
  ntree = 100, seed = 529
)
Proximity 
fit_proximity <- mrf3(
  tcga_brca, k = 4,
  main_clustering = "proximity",
  ntree = 100, seed = 529
)
fit_enhanced <- mrf3(
  tcga_brca, k = 4,
  main_clustering = "enhanced_proximity",
  ntree = 100, seed = 529
)

4.3. Branch-level controls

For finer control, use clustering_args inside mrf3_fit().

fit_branch <- mrf3_fit(
  dat.list = tcga_brca,
  ntree = 200,
  run_imd = TRUE,
  clustering_args = list(
    shared_method = "Spectral",
    specific_method = "PAM",
    specific_prox_method_cl = "PAM"
  ),
  seed = 529
)
Branch controls. shared_method: PAM or Spectral for the shared branch. specific_method: adds Proximity and Enhanced_Proximity for per-block clustering. specific_prox_method_cl: secondary clustering engine for proximity-based specific methods.

5. Variable Selection

5.1. Integrated workflow

Variable selection is driven by IMD weights. Enable it in the staged workflow:

fit_vs <- mrf3_fit(
  dat.list = tcga_brca,
  k = 4, ntree = 200,
  run_imd = TRUE,
  run_variable_selection = TRUE,
  variable_selection_args = list(
    method = "mixture",
    level = 0.05,
    normalized = TRUE
  ),
  seed = 529
)

get_selected_vars(fit_vs)
get_data(fit_vs, which = "selected")

Call mrf3_vs() directly on an existing fit to compare methods:

Filter

Test 
vs_test <- mrf3_vs(
  mod = fit, dat.list = tcga_brca,
  method = "test", re_fit = FALSE,
  level = 0.05
)
Mixture 
vs_mix <- mrf3_vs(
  mod = fit, dat.list = tcga_brca,

Inspect results

6. Robust Clustering

The robust branch first performs variable selection, then reclusters on the reduced representation:

fit_robust <- mrf3_fit(
  dat.list = tcga_brca,
  k = 4, ntree = 200,
  run_imd = TRUE,
  run_variable_selection = TRUE,
  run_cluster_imd = FALSE,
  run_robust_clustering = TRUE,
  variable_selection_args = list(method = "mixture"),
  seed = 529
)

table(get_clusters(fit_robust, which = "robust"))

7. Plotting Functions

All plot helpers accept an mrf3_fit object. The source argument selects which branch or reconstruction to visualize.

7.1. Embedding plots

plot_tsne(mod = fit, group = get_clusters(fit),
         source = "specific_shared",
         main = "t-SNE: shared/specific branch")

plot_umap(mod = fit_robust,
          group = get_clusters(fit_robust, which = "robust"),
          source = "robust_clustering",
          main = "UMAP: robust clustering")

7.2. Network

plot_network(dat = fit, group = get_clusters(fit),
             source = "specific_shared", cutoff = 0.3)

7.3. Feature importance

plot_weights(fit_vs, weight_source = "imd", top = 15)

7.4. Circos

plot_circos(mat = fit$reconstruction$W$W_all)

7.5. Survival analysis

clinical_ids <- tcga_brca_clinical$sampleID
clusters <- get_clusters(fit)
common_ids <- intersect(names(clusters), clinical_ids)
clinical_sub <- tcga_brca_clinical[
  match(common_ids, clinical_ids), , drop = FALSE
]

plot_km(test_var = clusters[common_ids],
        time_var = "OS.time", event_var = "OS",
        pheno_mat = clinical_sub)

8. Method Comparison Strategy

Compare clusters against PAM50 subtypes

Use the bundled clinical annotations to compare the three clustering backends (fitted in Section 4) against known PAM50 molecular subtypes:

pam50 <- tcga_brca_clinical$BRCA_Subtype_PAM50

# Pairwise ARI matrix (backends + PAM50)
cluster_metric_matrix(
  list(
    similarity         = get_clusters(fit),
    proximity          = get_clusters(fit_proximity),
    enhanced_proximity = get_clusters(fit_enhanced),
    PAM50              = pam50
  ),
  metric = "ari"
)

# Per-backend metrics vs PAM50
rbind(
  similarity         = cluster_metrics(get_clusters(fit), pam50),
  proximity          = cluster_metrics(get_clusters(fit_proximity), pam50),
  enhanced_proximity = cluster_metrics(get_clusters(fit_enhanced), pam50)
)

Recommended workflow

  1. Fit a baseline with mrf3(main_clustering = "similarity").
  2. Refit with alternative backends; compare against PAM50 using cluster_metric_matrix().
  3. Run filter, test, mixture on the same IMD weights.
  4. Visualize with plot_weights(), plot_heatmap(), plot_tsne().
  5. Promote shortlisted settings into run_robust_clustering = TRUE.

multiRF-sub

For larger blocks, the sub-MRF ensemble mode should be compared as another workflow variant rather than left out of the benchmark.

fit_sub <- mrf3_fit(
  dat.list = tcga_brca,
  k = 4,
  ntree = 200,
  sub_mrf = TRUE,
  sub_mrf_args = list(
    n_sub = 10,
    frac_response = 0.2,
    frac_predictor = 0.2,
    ntree_per_sub = 20
  ),
  run_imd = TRUE,
  seed = 529
)

summary(fit_sub)
table(get_clusters(fit_sub))
cluster_metrics(get_clusters(fit_sub)[common_ids], pam50)
