Getting Started with multiRF

Wei Zhang

2026-03-19

Overview

multiRF provides a workflow for integrating matched multi-omics datasets with random forests, MRF-style weighting, clustering, and downstream summaries.

The typical workflow is:

  1. Prepare a named list of omics matrices with matched samples.
  2. Run mrf3() to fit the integration model and produce cluster labels.
  3. Inspect weights, selected variables, and cluster assignments.
  4. Use plotting helpers for survival and embedding diagnostics.

Method Maps

Clustering choices

Setting Scope Typical use
similarity Global main_clustering Default fused-similarity workflow; best starting point.
proximity Global main_clustering Use when RF proximity is the main object of interest.
enhanced_proximity Global main_clustering Aggressive proximity-based branch.
PAM Shared or specific branch Medoid-based clustering.
Spectral Shared or specific branch Similarity geometry-based clustering.

Variable-selection choices

Method Signal model Typical use
filter Heuristic threshold Good baseline for a practical first pass.
test Significance-style Hypothesis-testing flavor for weight selection.
mixture Signal/noise mixture When weights separate into background and enriched components.

Plotting choices

Function Input focus
plot_tsne() Similarity or reconstruction
plot_umap() Similarity or reconstruction
plot_embed() Matrix-based quick embedding
plot_network() Similarity graph
plot_weights() IMD or cluster-IMD weights
plot_circos() Adjacency-like structure
plot_km() Cluster labels plus clinical
